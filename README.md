# 👁️🤖 Azure AI Services: Computer Vision

## 📖 Descripción

Proyecto de investigación y prueba de concepto (PoC) sobre las capacidades de Inteligencia Artificial de Microsoft Azure para el análisis de imágenes, reconocimiento facial y procesamiento de texto (OCR).

Este repositorio documenta el estudio realizado con **Azure AI Vision**, explorando cómo los sistemas en la nube permiten a una computadora interpretar contenido visual de manera similar a los humanos.

---

## 🛠️ Stack Tecnológico

* ☁️ **Cloud Platform:** Microsoft Azure (Vision Studio)
* 🧠 **Servicios de IA:** Azure AI Vision
* 📄 **Formato de salida:** JSON (metadatos y atributos detectados)

---

## 🚀 Capacidades Exploradas

Durante la investigación se validaron las siguientes funcionalidades:

### 🔍 Análisis de Contenido

* Generación automática de descripciones basadas en el contexto de la imagen

### 🧑‍🤝‍🧑 Reconocimiento de Objetos y Rostros

* Identificación de personas y objetos
* Detección de rostros
* Extracción de atributos faciales

### 🔤 Lectura de Texto (OCR)

* Extracción de texto dentro de imágenes
* Funcionamiento similar a herramientas como Google Lens

### 🧠 Interpretación de Contexto

* Comprensión de escenas completas (no solo objetos individuales)

---

## 📸 Evidencia de Implementación

El proyecto incluyó la configuración y uso del servicio en Azure:

* ⚙️ **Configuración del recurso:**

  * Selección de región

* 🙂 **Detección facial:**

  * Extracción de texto desde imágenes (ej: frases en imágenes)

## 📉 Retos y Lecciones Aprendidas

### ⚠️ Gestión de Suscripciones

* La falta de créditos o suscripciones deshabilitadas limita el acceso al servicio
* Importancia del control de costos en la nube

### 🌐 Dependencia de Infraestructura

* Requiere conexión estable a internet
* Dependencia de disponibilidad de Azure

### 🎯 Precisión del Modelo

* Menor exactitud en imágenes:

  * Borrosas
  * Oscuras
  * Con muchos objetos
